# Internationalization

ts-video-player supports multiple languages out of the box.

## Built-in Languages

The player includes translations for 6 languages:

- English (`en`)
- Spanish (`es`)
- French (`fr`)
- German (`de`)
- Japanese (`ja`)
- Chinese (`zh`)

```typescript
import {
  createPlayer,
  englishStrings,
  spanishStrings,
  frenchStrings,
  germanStrings,
  japaneseStrings,
  chineseStrings,
} from 'ts-video-player'

const player = createPlayer('#container', {
  src: '/video.mp4',
  i18n: {
    language: 'es',
    translations: spanishStrings,
  },
})
```

## Custom Translations

Create your own translations:

```typescript
import { createPlayer, type TranslationStrings } from 'ts-video-player'

const portugueseStrings: TranslationStrings = {
  play: 'Reproduzir',
  pause: 'Pausar',
  mute: 'Silenciar',
  unmute: 'Ativar som',
  enterFullscreen: 'Tela cheia',
  exitFullscreen: 'Sair da tela cheia',
  settings: 'Configurações',
  speed: 'Velocidade',
  quality: 'Qualidade',
  captions: 'Legendas',
  captionsOff: 'Desligado',
  seekForward: 'Avançar {0} segundos',
  seekBackward: 'Retroceder {0} segundos',
  played: 'Reproduzido',
  buffered: 'Armazenado',
  currentTime: 'Tempo atual',
  duration: 'Duração',
  volume: 'Volume',
  pip: 'Picture-in-Picture',
  airplay: 'AirPlay',
  googleCast: 'Google Cast',
  live: 'AO VIVO',
  goLive: 'Ir para o ao vivo',
  // ... more strings
}

const player = createPlayer('#container', {
  src: '/video.mp4',
  i18n: {
    language: 'pt',
    translations: portugueseStrings,
  },
})
```

## Using the I18n API

```typescript
import { createI18n, t } from 'ts-video-player'

// Create an i18n instance
const i18n = createI18n({
  language: 'es',
  translations: spanishStrings,
})

// Translate strings
console.log(t('play'))          // "Reproducir"
console.log(t('seekForward', 10)) // "Avanzar 10 segundos"
```

## Dynamic Language Switching

```typescript
import { createPlayer, setI18n, spanishStrings, englishStrings } from 'ts-video-player'

const player = createPlayer('#container', { src: '/video.mp4' })

// Switch language dynamically
function setLanguage(lang: string) {
  const translations = lang === 'es' ? spanishStrings : englishStrings
  setI18n({ language: lang, translations })
}

// Example: language selector
document.querySelector('#language-select')?.addEventListener('change', (e) => {
  setLanguage((e.target as HTMLSelectElement).value)
})
```
